Was ist Laravel?

- Opensurce PHP Framework -- Highlevel -- MVC designpattern -- 2011 -- opinionated Framework

Warum Laravel?

- Rapid develipment (Streamline common Tasks like Routing, Auth, database)
- Security (Built in Features for SQL-Injektion, XSS, CSRF, ect)
- Scalability 
- Built in Auth
- Elegant Syntax
- Robust ecosystem (Eloquent ORM, Blade Templates, Artisan CLI, Auth)
- Database Tools
- Community - Lange and active
